# Split The Bill – HackUPC 2025

Prototype of a bill-splitting feature inspired by Revolut, using OCR to extract receipt data and automate cost distribution.

https://d112y698adiu2z.cloudfront.net/photos/production/software_photos/003/406/294/datas/gallery.jpg

## Tech stack
- React-based frontend
- CSS for styling
- OCR backend 

## Key features
- Extract structured bill data from receipt images
- Assign bill items to individuals
- Calculate per-person payment amounts
- Support voice note input to interpret item assignments

## Project context
Built during HackUPC 2025 as a 36-hour hackathon project and submitted as part of Revolut’s sponsor challenge.
